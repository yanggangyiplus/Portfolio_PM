# DHAN Archive Rebuild

패션 브랜드 아카이브 웹사이트 기획

## 문제 정의

기존 HTML/CSS 기반 정적 사이트는 유지보수가 어려웠고, 시즌별 컬렉션 추가가 번거로웠습니다. 브랜드 아이덴티티를 효과적으로 표현할 수 있는 디자인 시스템이 필요했습니다.

## 해결 방법

1. **정보구조(IA) 설계**: 시즌별 컬렉션을 쉽게 관리할 수 있는 데이터 구조를 설계했습니다.
2. **동적 웹사이트 기획**: Next.js 기반 SSR 구조를 설계하여 빠른 초기 로딩과 동적 라우팅을 지원하도록 했습니다.
3. **디자인 시스템 구축**: 아이보리, 베이지, 차콜 톤의 브랜드 아이덴티티를 반영한 디자인 시스템을 구축했습니다.
4. **반응형 디자인 설계**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 경험을 제공하도록 설계했습니다.

## 성과

- Next.js 14 기반 SSR 구조 설계 완료
- 브랜드 아이덴티티를 효과적으로 표현하는 디자인 시스템 구축
- 시즌별 컬렉션을 쉽게 추가하고 관리할 수 있는 구조 설계
- 모든 기기에서 최적화된 경험을 제공하는 반응형 디자인 설계

## 주요 산출물

- **IA (정보구조도)**: 시즌별 컬렉션 데이터 구조 설계 (`seasons.json`)
- **UX Flow**: 페이지 라우팅 구조 및 사용자 플로우
- **Style Guide**: 색상 팔레트, 폰트, Glassmorphism 효과 정의
- **Screens**: 각 페이지별 디자인 구현
