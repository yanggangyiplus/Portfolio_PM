# UX Research and Improvement Project

사용자 연구 기반 UX 개선 프로젝트

## 프로젝트 개요

기존 서비스의 사용자 경험을 개선하기 위한 사용자 연구 및 UX 개선 프로젝트입니다. 사용자 인터뷰와 행동 분석을 통해 Pain Point를 도출하고, Before → After 개선안을 설계했습니다.

## 주요 기획 내용

### 사용자 연구
- 사용자 인터뷰 진행 및 분석
- 사용자 행동 패턴 분석
- Pain Point 도출 및 우선순위 설정
- 개선 기회 영역 식별

### UX 개선 설계
- Before → After 개선안 설계
- 사용자 플로우 개선안
- 프로토타입 제작
- 사용자 테스트 및 피드백 반영

## 산출물

- **User_Interviews**: 사용자 인터뷰 결과 및 인사이트
- **PainPoint_Report**: Pain Point 분석 보고서
- **UX_Flow**: 개선된 사용자 플로우
- **Before_After**: Before → After 개선안 비교
- **Prototype**: 개선안 프로토타입
- **Presentation**: 연구 결과 및 개선안 발표 자료

## 프로젝트 기간

2024.04 ~ 2024.08

## 역할

UX 리서처 / UX 디자이너 (UX Researcher / UX Designer)

## 주요 성과

- 사용자 인터뷰 10명 이상 진행 및 분석 완료
- 핵심 Pain Point 5개 도출
- Before → After 개선안 설계 완료
- 프로토타입 테스트를 통한 검증 완료
