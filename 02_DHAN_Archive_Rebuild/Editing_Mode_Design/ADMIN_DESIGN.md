# DHAN Archive - 관리자 편집 모드 설계

## 개요

D'HAN Archive 웹사이트의 관리자 편집 모드 설계 문서

## 관리자 모드 목적

- 시즌별 컬렉션 추가 및 수정
- 디자이너 노트 편집
- 이미지 및 영상 관리
- 브랜드 정보 업데이트

## 관리자 모드 구조

### 1. 로그인 및 인증
- 관리자 전용 로그인 페이지
- JWT 기반 인증
- 세션 관리

### 2. 대시보드
- 전체 컬렉션 현황
- 최근 수정 내역
- 빠른 액세스 메뉴

### 3. 컬렉션 관리

#### 시즌 추가/수정
- 시즌 ID, 제목, 부제목 입력
- 연도 선택
- 설명 입력
- 이미지 업로드 (다중 선택)
- 영상 업로드 (선택)

#### 디자이너 노트 편집
- 마크다운 에디터
- 미리보기 기능
- 저장 및 발행

#### 브랜드 런칭 쇼 관리 (2024 FW)
- 쇼 이미지 업로드
- 쇼 영상 업로드
- 썸네일 이미지 설정

### 4. 이미지 관리
- 이미지 업로드
- 이미지 삭제
- 이미지 순서 변경 (드래그 앤 드롭)
- 이미지 최적화 자동 처리

### 5. 브랜드 정보 관리
- 브랜드 비주얼 이미지 업로드
- 비전 텍스트 편집
- 프로필 정보 관리

## 사용자 인터페이스 설계

### 레이아웃
- 좌측 사이드바: 메뉴 네비게이션
- 메인 영역: 편집 폼 및 미리보기
- 우측 패널: 도움말 및 단축키 (선택)

### 주요 화면

#### 컬렉션 편집 화면
```
┌─────────────────────────────────────┐
│ [저장] [미리보기] [취소]            │
├─────────────────────────────────────┤
│ 시즌 정보                            │
│ - 시즌 ID: [2024FW]                  │
│ - 제목: [2024 F/W]                   │
│ - 부제목: [D'HAN]                    │
│ - 연도: [2024]                       │
│                                      │
│ 이미지 업로드                        │
│ [파일 선택] [드래그 앤 드롭]        │
│                                      │
│ 디자이너 노트                        │
│ [마크다운 에디터]                    │
└─────────────────────────────────────┘
```

### 인터랙션 설계

#### 이미지 업로드
- 드래그 앤 드롭 지원
- 다중 파일 선택
- 업로드 진행률 표시
- 이미지 미리보기

#### 텍스트 편집
- 실시간 미리보기
- 자동 저장 (드래프트)
- 버전 관리 (선택)

## 보안 고려사항

### 인증 및 권한
- 관리자 전용 로그인
- 세션 타임아웃 설정
- CSRF 토큰 사용

### 데이터 검증
- 입력 데이터 검증
- 파일 타입 및 크기 제한
- XSS 방지

## 사용자 경험 개선

### 자동 저장
- 편집 중 자동 저장 (드래프트)
- 실수로 닫아도 데이터 보존

### 미리보기
- 저장 전 미리보기 기능
- 실제 사이트와 동일한 스타일로 표시

### 단축키
- Ctrl+S: 저장
- Ctrl+P: 미리보기
- Esc: 취소
