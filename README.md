## 기획 역량 요약

### 서비스 기획
- **요구사항 분석**: 사용자 니즈 파악 및 비즈니스 요구사항 정의
- **기능명세서 작성**: FUNC-001부터 체계적으로 기능을 명세하여 작성
- **IA 설계**: 정보구조도 설계 및 ERD 작성
- **UX 플로우 설계**: 핵심 사용자 플로우 정의 및 예외 처리 설계

### UX 설계
- **와이어프레임 제작**: Figma를 활용한 와이어프레임 제작
- **사용자 플로우**: 핵심 플로우 다이어그램 작성
- **디자인 시스템**: 색상 팔레트, 폰트, 컴포넌트 가이드라인 정의
- **반응형 디자인**: 모바일/태블릿/데스크톱 최적화 설계

### 비즈니스 이해
- **시장 분석**: 경쟁사 분석 및 시장 동향 파악
- **비즈니스 모델 설계**: 수익 모델 검토 및 시장 분석
- **ROI 분석**: 비즈니스 모델 기반 수익성 검토
- **비즈니스 임팩트 측정**: 실제 배포 후 정량적/정성적 가치 분석

### 문서화
- **API 명세서**: RESTful API 엔드포인트 상세 명세
- **기능명세서**: 각 기능별 상세 명세서 작성
- **IA 다이어그램**: 정보구조도 및 ERD 다이어그램
- **UX 플로우 다이어그램**: 사용자 플로우 시각화
