## 📊 성과 그래프 생성 방법

### Excel/Google Sheets 사용
1. 데이터 입력 (Before, After 값)
2. 차트 생성 (막대 그래프 또는 선 그래프)
3. 차트를 이미지로 저장 (PNG 형식 권장)
4. images 폴더에 저장

### 온라인 도구 사용
- [Canva](https://www.canva.com/): 그래프 템플릿 사용
- [Chart.js](https://www.chartjs.org/): 웹 기반 차트 생성
