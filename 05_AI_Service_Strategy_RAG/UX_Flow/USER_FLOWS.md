# AI Service Strategy - 사용자 플로우

## 플로우 개요

RAG 기반 AI 서비스의 주요 사용자 플로우 설계

## 핵심 플로우

### 1. 문서 업로드 플로우
1. 문서 관리 페이지 접속
2. 문서 업로드 버튼 클릭
3. 파일 선택
4. 업로드 진행
5. 문서 처리 완료

### 2. 질의응답 플로우
1. 질의응답 페이지 접속
2. 질문 입력
3. 검색 실행
4. 관련 문서 표시
5. AI 답변 생성
6. 답변 확인

### 3. 문서 검색 플로우
1. 검색창에 키워드 입력
2. 검색 결과 표시
3. 관련 문서 선택
4. 문서 내용 확인
