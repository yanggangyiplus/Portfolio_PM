# Game Design - UI 와이어프레임

## UI 와이어프레임 개요

게임의 주요 UI 화면 와이어프레임 설계 문서

## 화면 목록

### 1. 메인 메뉴
- 게임 로고
- 시작 버튼
- 설정 버튼
- 상점 버튼
- 프로필 버튼

### 2. 게임 플레이 화면
- 게임 필드
- 점수 표시
- 시간 표시
- 일시정지 버튼
- 힌트 버튼

### 3. 스테이지 선택 화면
- 스테이지 맵
- 스테이지 정보
- 클리어 상태 표시
- 보상 미리보기

### 4. 인벤토리 화면
- 아이템 목록
- 캐릭터 정보
- 장비 슬롯
- 강화 버튼

### 5. 상점 화면
- 상품 목록
- 가격 표시
- 구매 버튼
- 할인 정보

## UI 설계 원칙

### 1. 직관성
- 명확한 아이콘 사용
- 일관된 디자인 언어
- 쉬운 네비게이션

### 2. 가시성
- 중요한 정보 강조
- 적절한 색상 대비
- 명확한 버튼 크기

### 3. 접근성
- 터치 친화적 크기
- 명확한 피드백
- 오류 방지

## 주요 인터랙션

### 버튼 인터랙션
- 터치 시 하이라이트
- 클릭 피드백
- 로딩 상태 표시

### 메뉴 전환
- 부드러운 애니메이션
- 명확한 전환 효과
- 빠른 응답 시간

## 모바일 최적화

### 레이아웃
- 세로 모드 우선
- 터치 영역 최소 44x44px
- 안전 영역 고려

### 성능
- 최적화된 이미지
- 효율적인 렌더링
- 배터리 소모 최소화
