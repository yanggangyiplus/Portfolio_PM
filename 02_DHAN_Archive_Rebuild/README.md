# DHAN Archive Rebuild

패션 브랜드 아카이브 웹사이트 기획

## 프로젝트 개요

디자이너 브랜드 D'HAN의 컬렉션을 아카이브하고 소개하는 웹사이트 재구축 기획 프로젝트입니다. 브랜드 아이덴티티를 반영한 디자인 시스템을 구축하고, 시즌별 컬렉션을 효과적으로 전시할 수 있는 구조를 설계했습니다.

## 주요 기획 내용

### 정보 구조 설계
- 시즌별 컬렉션 구조 설계
- 브랜드 스토리 및 비전 페이지 설계
- 디자이너 프로필 페이지 설계
- 동적 라우팅 구조 설계

### 디자인 시스템
- 브랜드 컬러 팔레트 정의 (아이보리, 베이지, 차콜)
- 타이포그래피 가이드라인
- 컴포넌트 디자인 시스템
- 반응형 디자인 가이드

### UX 설계
- 시즌별 타임라인 UX 플로우
- 갤러리 형식 콘텐츠 구성
- 모바일/태블릿/데스크톱 최적화

## 산출물

- **IA**: 정보 구조도 및 데이터 구조 설계
- **UX_Flow**: 사용자 플로우 다이어그램
- **Style_Guide**: 디자인 시스템 가이드라인
- **Screens**: 주요 화면 디자인
- **Editing_Mode_Design**: 관리자 편집 모드 설계
- **Presentation**: 기획 발표 자료

## 프로젝트 기간

2024.07 ~ 2024.12

## 역할

기획자 / UX 디자이너 (Product Planner / UX Designer)

## 주요 성과

- 브랜드 아이덴티티를 반영한 디자인 시스템 구축
- Next.js 기반 동적 웹사이트 구조 설계
- 반응형 디자인 가이드라인 수립
- 시즌별 컬렉션 관리 구조 설계 완료
