# DHAN Archive Rebuild - 기획 발표 자료

## 프로젝트 개요

패션 브랜드 D'HAN의 컬렉션을 아카이브하고 소개하는 웹사이트 재구축 기획 프로젝트

## 발표 목차

1. 프로젝트 배경 및 목적
2. 브랜드 아이덴티티 분석
3. 정보 구조 설계
4. 디자인 시스템 구축
5. 사용자 경험 설계
6. 기술 스택 및 구현 방안
7. 기대 효과

## 1. 프로젝트 배경 및 목적

### 기존 사이트의 문제점
- 정적 HTML/CSS 기반으로 유지보수 어려움
- 시즌별 컬렉션 추가가 번거로움
- 모바일 최적화 부족
- 브랜드 아이덴티티 표현 한계

### 재구축 목적
- 동적 웹사이트로 전환하여 관리 효율성 향상
- 브랜드 아이덴티티를 반영한 디자인 시스템 구축
- 반응형 디자인으로 모든 기기에서 최적화된 경험 제공
- 시즌별 컬렉션을 체계적으로 아카이브

## 2. 브랜드 아이덴티티 분석

### 브랜드 철학
**"Embrace Leisure, Live Busy"**

D'HAN은 움직임 속의 정제된 감정을 옷으로 표현하는 디자이너 브랜드입니다.

### 브랜드 키워드
- 감정의 형태화
- 스포츠의 리듬과 우아함
- 섬세한 디테일
- 균형 잡힌 실루엣

### 디자인 방향성
- 미니멀리즘
- 감성적 미학
- 정제된 에너지
- 우아함의 균형

## 3. 정보 구조 설계

### 페이지 구조
```
홈 (/)
├── 브랜드 소개
├── 시즌별 컬렉션 타임라인
└── 프로필 링크

시즌별 컬렉션 (/[season])
├── 컬렉션 이미지 갤러리
├── 디자이너 노트
└── 브랜드 런칭 쇼 (2024 FW)

브랜드 (/brand)
├── 브랜드 비주얼
└── 비전

프로필 (/profile)
├── 디자이너 정보
└── 수상 내역
```

### 데이터 구조
- 시즌별 컬렉션 메타데이터
- 이미지 및 영상 관리
- 디자이너 노트 관리

## 4. 디자인 시스템 구축

### 컬러 팔레트
- 아이보리 (#F5F5DC): 배경 및 여백
- 베이지 (#F0EBE0): 보조 배경
- 차콜 (#36454F): 텍스트 및 강조

### 타이포그래피
- 제목: Serif 폰트 (감성적 표현)
- 본문: Sans-serif 폰트 (가독성)

### 컴포넌트
- 헤더/푸터
- 이미지 갤러리
- 텍스트 블록
- 버튼 및 링크

## 5. 사용자 경험 설계

### 핵심 사용자 플로우

#### 컬렉션 탐색 플로우
1. 홈페이지 접속
2. 시즌별 타임라인 확인
3. 관심 있는 시즌 클릭
4. 컬렉션 이미지 갤러리 탐색
5. 디자이너 노트 읽기

#### 브랜드 스토리 탐색 플로우
1. 브랜드 메뉴 클릭
2. 브랜드 비주얼 확인
3. 비전 읽기

### 반응형 디자인
- 모바일: 단일 컬럼, 터치 최적화
- 태블릿: 2컬럼 레이아웃
- 데스크톱: 3컬럼 레이아웃, 넓은 화면 활용

## 6. 기술 스택 및 구현 방안

### 프론트엔드
- Next.js 14: SSR 및 동적 라우팅
- React 18: 컴포넌트 기반 개발
- Tailwind CSS: 유틸리티 기반 스타일링

### 데이터 관리
- JSON 기반 정적 데이터
- 이미지 최적화 및 CDN 활용

### 배포
- Vercel: Next.js 최적화 배포 플랫폼
- 자동 배포 파이프라인

## 7. 기대 효과

### 사용자 측면
- 브랜드 아이덴티티를 명확히 전달
- 모든 기기에서 최적화된 경험 제공
- 직관적인 네비게이션으로 컬렉션 탐색 용이

### 비즈니스 측면
- 웹사이트 관리 효율성 향상
- 시즌별 컬렉션 추가 용이
- 브랜드 이미지 강화

## 결론

브랜드 아이덴티티를 반영한 디자인 시스템과 체계적인 정보 구조를 통해 D'HAN 브랜드를 효과적으로 전달할 수 있는 웹사이트 재구축 기획안을 제시했습니다.
