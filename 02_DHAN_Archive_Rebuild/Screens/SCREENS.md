# DHAN Archive - 주요 화면 디자인

## 화면 목록

### 1. 홈페이지 (메인)
- 브랜드 소개 섹션
- 시즌별 컬렉션 타임라인
- 최신 시즌 하이라이트

### 2. 시즌별 컬렉션 페이지
- 컬렉션 이미지 갤러리 (3컬럼 그리드)
- 디자이너 노트 섹션
- 브랜드 런칭 쇼 (2024 FW만)

### 3. 브랜드 페이지
- 브랜드 비주얼 이미지
- 비전 텍스트

### 4. 프로필 페이지
- 디자이너 프로필 사진
- 기본 정보 (생년월일, 이메일, 인스타그램)
- 수상 내역 (연도별 정리)

## 디자인 상세

### 홈페이지
- 배경: 아이보리/베이지 톤
- 타이포그래피: Serif 폰트로 브랜드 감성 표현
- 레이아웃: 중앙 정렬, 넓은 여백

### 컬렉션 페이지
- 이미지 갤러리: 3컬럼 그리드 (데스크톱)
- 이미지 호버 효과: 약간의 확대 및 그림자
- 디자이너 노트: 좌측 정렬, 읽기 편한 폰트 크기

### 브랜드 페이지
- 브랜드 비주얼: 전체 너비 이미지
- 비전: 중앙 정렬, 큰 폰트 크기

### 프로필 페이지
- 프로필 사진: 원형, 중앙 배치
- 정보: 좌측 정렬, 명확한 레이블
- 수상 내역: 카드 형식, 연도별 그룹화

## 반응형 디자인

### 모바일 (< 768px)
- 단일 컬럼 레이아웃
- 이미지 갤러리: 1컬럼
- 터치 친화적 버튼 크기

### 태블릿 (768px ~ 1024px)
- 2컬럼 레이아웃
- 이미지 갤러리: 2컬럼

### 데스크톱 (> 1024px)
- 3컬럼 레이아웃
- 이미지 갤러리: 3컬럼
- 넓은 화면 활용

## 색상 및 타이포그래피

### 색상 팔레트
- 아이보리: #F5F5DC (배경)
- 베이지: #F0EBE0 (보조 배경)
- 차콜: #36454F (텍스트)

### 타이포그래피
- 제목: Serif 폰트, 32-48px
- 본문: Sans-serif 폰트, 16px
- 작은 텍스트: 14px

## 인터랙션

### 이미지 갤러리
- 호버 시 약간 확대
- 클릭 시 전체 화면 보기 (선택)
- 부드러운 전환 애니메이션

### 네비게이션
- 스크롤 시 헤더 고정
- 부드러운 스크롤 애니메이션
- 활성 메뉴 하이라이트
