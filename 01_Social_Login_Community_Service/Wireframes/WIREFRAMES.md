# Social Login Community Service - 와이어프레임

## 와이어프레임 개요

소셜 로그인 기반 커뮤니티 플랫폼의 핵심 화면 와이어프레임

## 화면 목록

### 1. 메인 페이지 (홈)
- 헤더: 로고, 네비게이션, 로그인 버튼
- 메인 영역: 인기 게시글, 최신 게시글 목록
- 사이드바: 카테고리 목록, 인기 사용자
- 푸터: 서비스 정보, 이용약관

### 2. 로그인 페이지
- 소셜 로그인 버튼 (카카오, 네이버, 구글)
- 간단한 서비스 소개
- 이용약관 동의 체크박스

### 3. 게시글 목록 페이지
- 카테고리 필터
- 정렬 옵션 (최신순, 인기순, 댓글순)
- 게시글 카드 리스트
- 페이지네이션

### 4. 게시글 작성 페이지
- 카테고리 선택 드롭다운
- 제목 입력 필드
- 본문 작성 에디터
- 첨부파일 업로드 영역
- 작성/취소 버튼

### 5. 게시글 상세 페이지
- 게시글 제목 및 내용
- 작성자 정보 및 작성 시간
- 좋아요, 댓글 수 표시
- 댓글 영역
- 관련 게시글 추천

### 6. 사용자 프로필 페이지
- 프로필 이미지 및 기본 정보
- 작성한 게시글 목록
- 댓글 작성 내역
- 좋아요한 게시글

### 7. 관리자 대시보드
- 통계 요약 카드
- 최근 신고 내역
- 사용자 관리
- 게시글 관리

## 와이어프레임 설계 원칙

### 1. 사용자 중심 설계
- 직관적인 네비게이션
- 명확한 정보 구조
- 최소한의 클릭으로 원하는 기능 접근

### 2. 반응형 디자인
- 모바일 우선 설계
- 태블릿 및 데스크톱 최적화
- 터치 친화적 인터페이스

### 3. 접근성 고려
- 명확한 버튼 레이블
- 충분한 색상 대비
- 키보드 네비게이션 지원

## 주요 인터랙션

### 로그인 플로우
1. 메인 페이지에서 "로그인" 버튼 클릭
2. 소셜 로그인 선택 화면 표시
3. 소셜 계정 선택 및 인증 진행
4. 자동 회원가입/로그인 완료 후 메인 페이지로 이동

### 게시글 작성 플로우
1. "글쓰기" 버튼 클릭
2. 카테고리 선택
3. 제목 및 본문 작성
4. 첨부파일 추가 (선택)
5. "등록" 버튼 클릭
6. 작성 완료 후 게시글 상세 페이지로 이동

### 댓글 작성 플로우
1. 게시글 하단 댓글 입력 영역 클릭
2. 댓글 내용 입력
3. "등록" 버튼 클릭
4. 댓글 목록에 즉시 반영

## 디자인 가이드라인

### 레이아웃
- 최대 너비: 1200px
- 그리드 시스템: 12컬럼
- 여백: 16px 기본 단위

### 타이포그래피
- 제목: 24px, Bold
- 본문: 16px, Regular
- 작은 텍스트: 14px, Regular

### 색상
- Primary: #007BFF (파란색)
- Secondary: #6C757D (회색)
- Success: #28A745 (초록색)
- Danger: #DC3545 (빨간색)
