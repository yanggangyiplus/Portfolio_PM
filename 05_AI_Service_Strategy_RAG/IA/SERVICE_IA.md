# AI Service Strategy - 서비스 정보 구조도

## 정보 구조 개요

RAG 기반 AI 서비스의 정보 구조 설계

## 페이지 구조

```
홈 (/)
├── 서비스 소개
├── 주요 기능
└── 시작하기

문서 관리 (/documents)
├── 문서 업로드
├── 문서 목록
└── 문서 검색

질의응답 (/query)
├── 질문 입력
├── 검색 결과
└── 답변 생성

설정 (/settings)
├── 계정 설정
├── 문서 관리 설정
└── 알림 설정
```

## 데이터 구조

### 문서 관리
- 문서 메타데이터
- 문서 내용
- 임베딩 벡터

### 질의응답
- 질문
- 검색 결과
- 생성된 답변
