# Social Login Community Service

소셜 로그인 기반 커뮤니티 플랫폼 기획

![프로젝트 대표 이미지](./images/project-overview.png)
*이미지 추가 예정: 프로젝트 대표 화면 또는 와이어프레임*

## 프로젝트 개요

카카오, 네이버, 구글 소셜 로그인을 통합한 커뮤니티 플랫폼 기획 프로젝트입니다. 사용자 편의성을 높이고 회원 가입 장벽을 낮추기 위한 소셜 로그인 통합 및 커뮤니티 기능을 설계했습니다.

## 주요 기획 내용

### 소셜 로그인 통합
- 카카오 OAuth 2.0 연동 설계
- 네이버 OAuth 2.0 연동 설계
- 구글 OAuth 2.0 연동 설계
- 통합 인증 플로우 설계

### 커뮤니티 기능
- 게시글 작성 및 조회 기능
- 댓글 및 답글 기능
- 좋아요 및 북마크 기능
- 사용자 프로필 관리

## 산출물

- **IA**: 서비스 정보 구조도 및 ERD
- **UX_Flow**: 사용자 플로우 다이어그램 (로그인, 게시글 작성, 신고 등)
- **Wireframes**: 핵심 화면 와이어프레임
- **Requirements**: 기능명세서 (FUNC-001 ~ FUNC-005)
- **Survey_Research**: 사용자 조사 및 시장 분석
- **Roadmap**: 서비스 로드맵 및 우선순위
- **Presentation**: 기획 발표 자료

## 프로젝트 기간

2024.03 ~ 2024.06

## 역할

기획자 (Product Planner)

## 주요 성과

### 기획 역량
- 소셜 로그인 통합 플로우 설계 완료
- 커뮤니티 핵심 기능 5개 기능명세서 작성
- 사용자 플로우 3개 설계 (로그인, 게시글 작성, 신고)
- IA 설계 및 ERD 작성 완료
- **실제 의사결정 과정 문서화**: 기술 스택 선택, 대안 검토, 트레이드오프 분석

### 실행력
- **프로토타입 제작 완료**: Figma 기반 인터랙티브 프로토타입
- **기술 아키텍처 설계**: 실제 개발 가능한 수준의 상세 설계
- **기능 명세서 작성**: 개발팀 전달 가능한 수준의 상세 명세

### 비즈니스 이해
- **비즈니스 모델 설계**: 프리미엄 구독, 광고, 데이터 분석 수익 모델 검토
- **시장 분석**: 시장 규모 분석 및 타겟 시장 정의
- **수익성 분석**: 비즈니스 모델 기반 수익성 검토 (가정치)
- **경쟁 분석**: 카카오톡 오픈채팅, 네이버 카페, 디스코드 비교 분석

### 데이터 분석
- **시장 조사**: 경쟁사 분석 및 시장 동향 파악 (공개 자료 기반)
- **경쟁사 분석**: 주요 커뮤니티 플랫폼 조사
- **시장 분석 기반 기획**: 시장 조사 결과를 바탕으로 한 기능 우선순위 설정
